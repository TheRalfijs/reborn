# Ralfijs Reborn

A private personal discipline and progress-tracking web app.

## How to use

1. Open `index.html` in Chrome, Safari, or Edge.
2. Complete daily tasks and save your daily check-in.
3. Open Progress and enter your monthly performance numbers.
4. Open Monthly to compare growth or regression.
5. Use Settings → Export backup to save your data.

## Data privacy

All data is stored in your browser using localStorage. Nothing is uploaded anywhere.

## Important

If browser data is cleared, your saved entries may be deleted. Export a backup regularly.
